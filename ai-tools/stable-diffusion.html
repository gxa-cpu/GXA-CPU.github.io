🎨 Stable Diffusion (WebUI)

分类：图片类
🔗 官方网站：https://github.com/AUTOMATIC1111/stable-diffusion-webui

💡 用途介绍

Stable Diffusion 是开源的 AI 绘画模型，最大的优势是免费、可本地运行、可自由定制。通过 WebUI 界面（也叫 AUTOMATIC1111），你可以像用 Midjourney 一样输入文字生成图片，而且完全不用付费，生成次数无限制。还有海量社区模型和 LoRA 可以切换不同画风。

💰 收费情况

完全免费

完全免费开源！只需一台有独立显卡的电脑（建议 8GB 显存以上）。也可以用免费的云服务（如 Google Colab）运行。

📝 注册步骤（手把手教）

  • 1. 准备环境 —— 安装 Python 3.10 和 Git
  • 2. 克隆项目 —— 打开命令行，输入 git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
  • 3. 下载模型 —— 去 civitai.com 下载模型文件（.safetensors），放到 models/Stable-diffusion/ 目录下。新手推荐下载 SDXL 或 DreamShaper 模型
  • 4. 启动程序 —— 进入项目目录，运行 webui-user.bat（Windows）或 ./webui.sh（Mac/Linux）
  • 5. 等待加载 —— 首次启动会自动安装依赖，大约 5-10 分钟
  • 6. 打开浏览器 —— 程序启动后会显示一个网址（通常是 127.0.0.1:7860），在浏览器中打开即可
  • 7. 开始使用 —— 界面已经准备好了，直接输入提示词就能生成图片

🖥️ Stable Diffusion WebUI 界面介绍

📋 txt2img（文生图）标签页

  • Prompt（正向提示词）：描述你想要什么，例如「a beautiful sunset over the ocean, 4K」
  • Negative prompt（反向提示词）：描述你不想要什么，例如「blurry, ugly, deformed」
  • Sampling method（采样方法）：推荐 Euler a 或 DPM++ 2M Karras
  • Sampling steps（采样步数）：推荐 20-30 步，越多越精细但越慢
  • Resolution（分辨率）：设置图片宽高，推荐 1024×1024
  • Batch size（批量）：一次生成几张图，建议 2-4
  • 「Generate」生成按钮：点击开始生成

📋 img2img（图生图）标签页

  • 上传参考图：拖入或选择一张图片作为基础
  • Denoising strength：控制变化程度，0.3 微调、0.7 大改
  • 其余参数和 txt2img 类似

📋 其他功能标签页

  • Extras：放大图片分辨率（高清化）
  • PNG Info：上传图片查看其生成参数
  • Checkpoint Merger：合并两个模型
  • Extensions：安装各种插件扩展功能

🚀 操作步骤（跟着做就行）

  • 第1步：选择模型 —— 页面左上角的下拉框选择你下载的模型（如 DreamShaper）
  • 第2步：输入提示词 —— 在 Prompt 框输入你的描述，例如：「masterpiece, best quality, 1girl, standing in a flower garden, soft sunlight」
  • 第3步：输入反向提示词 —— 在 Negative prompt 框输入排除内容，例如：「lowres, bad anatomy, bad hands, blurry」
  • 第4步：调整参数 —— 采样步数设 25，分辨率设 1024×1024，其他保持默认
  • 第5步：点击 Generate —— 等待 10-30 秒，图片就会出现在右侧
  • 第6步：保存图片 —— 右键图片选择保存，或点击图片下方的保存按钮

👥 适合什么人

有电脑基础的人：设计师、插画师、AI 绘画爱好者、独立开发者。需要有一点电脑操作能力来安装软件，但生成图片本身很简单。适合不想花钱又想无限生成的人。
